# BNCC Event Feedback System

Website sederhana untuk mengumpulkan feedback event BNCC.
Dibuat untuk memenuhi tugas Quest Rolling Week RnD 37.

## Fitur

- **Public:** Form input feedback (Nama, Event, Rating, dll).
- **Admin:** Dashboard untuk melihat dan menghapus data feedback.

## Cara Menjalankan (Local)

1. Clone repository ini:

   git clone https://github.com/OlanIsm/quest-bncc-group.git https://github.com/OlanIsm/quest-bncc-group.git

2. Masuk ke folder
   cd quest-bncc-group

3. Install dependencies
   npm install

4. Jalankan server:
   node server.js
